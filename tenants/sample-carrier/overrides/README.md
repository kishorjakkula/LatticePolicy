Sample Carrier Overrides

Scope
- Use these folders for allowed extensions only. Do not modify core or product pack files.

Possible Overrides (low-code preferred)
- rating/ (adjust factors, tables, or provide a small hook to post-adjust ratings)
- underwriting/ (eligibility/referral rules in declarative form)
- documents/ (template mapping and conditions)

Notes
- Keep overrides minimal and declarative where possible to reduce upgrade friction.
- Follow the config merge order described in `docs/ARCHITECTURE.md`.
